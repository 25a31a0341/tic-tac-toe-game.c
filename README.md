Tic Tac Toe Game in C

Description

This is a simple console-based Tic Tac Toe game written in C. It allows two players to play the game by taking turns and entering their moves via the keyboard.

---

Features

* Two-player mode (Player X and Player O)
* Simple and interactive console interface
* Input validation for correct moves
* Detects win and draw conditions

---

Technologies Used

* C Programming Language
* Standard Input/Output (stdio.h)

---
 How to Run

1. Save the file as `tic_tac_toe.c`
2. Open terminal or command prompt
3. Compile the code:

   ```bash
   gcc tic_tac_toe.c -o tic_tac_toe
   ```
4. Run the program:

bash
   ./tic_tac_toe


How to Play

* The game is played on a 3x3 grid
* Player 1 uses **X**
* Player 2 uses **O**
* Players take turns entering a number (1–9)
* First player to align 3 marks wins
* If all positions are filled → Draw
